This is a port of TheOfficialFloW's BBS Vita plugin to PPSSPP, so credits to him!
https://github.com/TheOfficialFloW/RemasteredControls


REQUIREMENTS
·Latest PPSSPP version (1.11 or higher).

INSTRUCTIONS
1- Copy memstick folder to your PPSSPP folder, say overwrite if prompted.
2- Abre el emulador PPSSPP, y en Settings -> Controls -> Control mapping, configure your pad:
	·RightAn.Up.
	·RightAn.Down.
	·RightAn.Left.
	·RightAn.Right.
3- Enjoy!



REQUISITOS
·Última versión de PPSSPP (1.11 o superior).

INSTRUCCIONES
1- Copiar la carpeta memstick a tu carpeta PPSSPP, sobreescribe los archivos si es necesario.
2- Abre el emulador PPSSPP, y en Ajustes -> Controles -> Asignar botones, configura con tu mando:
	·StickDer.arrib.
	·StickDer.abaj.
	·StickDer.izq.
	·StickDer.der.
3- ¡A disfrutar!
